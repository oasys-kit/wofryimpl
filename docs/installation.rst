Installation
============

Install from PyPI::

    pip install wofryimpl

Install from source::

    git clone https://github.com/oasys-kit/wofryimpl
    cd wofryimpl
    pip install -e .

Dependencies
------------

* `wofry <https://github.com/oasys-kit/wofry>`_
* `syned <https://github.com/oasys-kit/syned>`_
* `srxraylib <https://github.com/oasys-kit/srxraylib>`_
* `pySRU <https://github.com/oasys-kit/pySRU>`_
* `dabax <https://github.com/oasys-kit/dabax>`_
* `numba <https://numba.pydata.org/>`_
